# Delle — NixOS Configuration

Dell Precision M2800 | i7-4810MQ | 15.6GB RAM | 256GB SSD

## Structure

```sh
/etc/nixos/
├── flake.nix
├── hosts/
│   └── delle/
│       ├── default.nix          # top-level host config
│       ├── disko.nix            # declarative partition layout
│       └── hardware.nix         # generated by nixos-generate-config (placeholder)
└── modules/
    ├── core.nix                 # nix settings, locale, time, fonts
    ├── desktop.nix              # niri, greetd/regreet, wayland, graphics, audio
    ├── hardware.nix             # bluetooth, power, intel graphics
    ├── networking.nix           # networkmanager, tailscale, firewall
    ├── services.nix             # openclaw, docker, podman, ssh
    ├── dev.nix                  # dev tools, vscode, claude-code
    └── users.nix                # craole (main) + cc (system/service)
```

## Install Steps

1. Boot the minimal ISO on Delle
2. Copy this config to `/etc/nixos/`
3. Run `lsblk` to confirm disk is `/dev/sda` — update `disko.nix` if not
4. Run: `sudo nix --experimental-features "nix-command flakes" run github:nix-community/disko -- --mode disko /etc/nixos/hosts/delle/disko.nix`
5. Run: `sudo nixos-generate-config --root /mnt` then copy `hardware-configuration.nix` to `hosts/delle/hardware.nix`
6. Run: `sudo nixos-install --flake /etc/nixos#delle`
7. Set password for `craole` when prompted
8. Reboot

## Post-Install

- Run `sudo tailscale up` to authenticate Tailscale
- Run `openclaw onboard` to set up the OpenClaw gateway
- Run `claude` to authenticate Claude Code
